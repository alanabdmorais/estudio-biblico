# 📚 Estúdio Bíblico - Plataforma Comunitária

Plataforma web para estudos bíblicos colaborativos, onde usuários podem criar, editar e compartilhar conteúdo comunitário.

## 🚀 Funcionalidades

- ✅ Autenticação de usuários (Nome + Senha)
- ✅ Recuperação de senha por validação de dados pessoais
- ✅ Painel administrativo
- ✅ Criação de capítulos, títulos e subtítulos
- ✅ Referências bíblicas com busca automática
- ✅ IA integrada (Groq) para contexto histórico e teológico
- ✅ Biblioteca comunitária com permissões
- ✅ Exportação para Word e PDF
- ✅ Imagens e complementos multimídia

## 🛠️ Tecnologias

- HTML5 / CSS3 / JavaScript (Vanilla)
- LocalStorage (persistência de dados)
- Groq API (LLM para contexto bíblico)
- Bible API (busca de versículos)
- GitHub Pages / Netlify (hospedagem)

## 📦 Instalação

```bash
git clone https://github.com/alanabdmorais/estudio-biblico.git
cd estudio-biblico
```

Abra o arquivo `index.html` no navegador.
